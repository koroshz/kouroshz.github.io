CIE

Causal Inference Enrichment (CIE) CIE is a platform for inference of active transcriptional regulators of differential gene expression. CIE provides a user-friendly we\ b-app and an R-package to run inference queries on casual regulatory netowrks. Several regulatory networks are provided in the platform, including a TF-gene intraction network c\ onstructed from ChIP-seq and tissue specific gene expression data, a netowrk based on STRING-DB, and curated TF-gene interaction networks such as TRRUST and TRED. The R-package \ provides functionality for custom networks as well. The R package can be obtained from here here.

FENN

FENN is an R package implementing a Von Neumann Entropy based distance metric learning model. The method identifies informative and non-informative features in the dat\ a and projects the data onto directions of maximal dissimilarity. The method can be used for supervised and semi-supervised applications as well as high dimensional data visuali\ zation.

creNET

creNET is an R package developed for prediction of response or risk factors using base-line gene expession data. The method is a biological network-based regularizatio\ n model for regression and classification.

openCage

OpenCage is an open source computer vision based software developed in C++ for quantifying mouse home-cage behaviors.

RNAsc

RNAsc is an open source software written in C++ that uses chemical probing data and determines the RNA secondary structure with high accuracy.
